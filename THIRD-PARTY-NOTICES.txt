# Third-Party Notices

`provgate` is licensed under the Apache License, Version 2.0 (see [`LICENSE`](LICENSE) and
[`NOTICE`](NOTICE)).

The deployed service (the container image built from [`Dockerfile`](Dockerfile)) bundles
the third-party Python components below: the runtime dependencies from
[`pyproject.toml`](pyproject.toml) plus their transitive dependencies, as pinned in
[`uv.lock`](uv.lock). Dev-only tooling (`pytest`, `respx`, `ruff`, `mypy`, `types-pyyaml`)
is not shipped and is not listed. Each license and copyright comes from the installed
package's `*.dist-info/METADATA` and bundled `LICENSE`. None of these packages ship a
separate `NOTICE` file.
