# Third-Party Notices

`provgate` is licensed under the Apache License, Version 2.0 (see [`LICENSE`](LICENSE) and [`NOTICE`](NOTICE)).

The deployed service (the container image built from [`Dockerfile`](Dockerfile)) incorporates the third-party
Python components listed below. These are the **runtime (production) dependencies only** — i.e. the packages
pulled in by `[project].dependencies` in [`pyproject.toml`](pyproject.toml) and their transitive dependencies,
as resolved in [`uv.lock`](uv.lock). Development-only tooling (`pytest`, `respx`, `ruff`, `mypy`,
`types-pyyaml`, and their own transitive dependencies) is not shipped in the deployed service and is not
listed here.

Each entry's license and copyright/author information was read directly from the installed package's
`*.dist-info/METADATA` file (and bundled `LICENSE` file, where present) at the pinned version in `uv.lock`.
None of the packages below ship a separate `NOTICE` file, so there is no additional attribution text to
reproduce beyond what appears here.

This file is generated by reading real package metadata, not by generic assumption. If a dependency's version
changes, regenerate the corresponding entry from the newly installed package's metadata rather than editing
the version number in place.
